# Wetube Reloaded(Routers)

/_ Global Routers _/
[] / -> Home
[] /join -> Join
[] /login -> Login
[] /search -> Search

/_ User Routers _/
[] /users/:id -> See User
[] /users/logout -> Log Out
[] /users/edit -> Edit My Profile
[] /users/remove -> Delete My Profile

/_ Video Routers _/
[] /videos/:id -> See video
[] /videos/:id/edit -> Edit video
[] /videos/:id/delete -> Delete video
[] /videos/upload -> Upload Video
